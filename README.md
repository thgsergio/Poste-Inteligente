# Poste-Inteligente
O Projeto Poste Inteligente consiste numa solução sustentável para os problemas de iluminação pública. Com o auxílio de Arduino, sensores e painéis solares, o objetivo esse projeto é monitorar a qualidade do ar da região e propor o desenvolvimento de postes que utilizam energia solar para seu funcionamento. Energia sustentável é o futuro do mundo!
